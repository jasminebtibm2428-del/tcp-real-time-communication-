# TCP Real-Time Communication (Python Sockets)

This project demonstrates simple real-time communication between a server and client using TCP sockets in Python.

## Files
- **server.py** — Runs a TCP server, waits for client, and exchanges messages.
- **client.py** — Connects to the server and allows two-way chat.

## How to Run
### 1. Start Server
```bash
python server.py
